bmultisig is a [bcoin][bcoin] plugin which adds multisig wallet http endpoints and events to bwallet for payment proposals.

## Note
**This is work in progress, not ready for production use.**

## Usage

``` bash
$ npm install -g bmultisig
$ bcoin --no-wallet
$ bmultisig
```
